# PCG-Game
This project is a third-person game developed in Unreal Engine.
The game focuses on procedural room generation and room-based enemy encounters implemented using Blueprint visual scripting.
# Procedural Generation
Rooms are generated dynamically using modular room template.
This ensures varied level layouts across different playthroughs.
